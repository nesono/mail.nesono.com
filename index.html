Nesono Mail Server Help

This is the official help for the Nesono Mail Server. We provide basic documentation around mail client configuration. The server provides IMAPS, SMTPS, and Sieve (server-based mail filtering) access.

Known Limitations

This is the list of known limitations including potential mitigations
Service Limitation Mitigation
Roundcube Sent messages folder default is not 'Sent Messages' but 'Sent' You can change the folder in Roundcube settings: 
Settings -> Preferences -> Special Folders
Roundcube Subscribed folders are not necessarily aligned with other mail clients You can change the folder in Roundcube settings: 
Settings -> Preferences -> Special Folders
as well as in 
Settings -> Folders
